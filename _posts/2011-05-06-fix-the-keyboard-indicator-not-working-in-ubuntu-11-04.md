in gconf-editor, check **/desktop/gnome/peripherals/keyboard/indicator/showFlags**

install set of flags using 
```bash
sudo apt-get install famfamfam-flag-png
```

make a softlink: 
```bash
ln -s /usr/share/flags/countries/16&#215;11 ~/.icons/flags
```
